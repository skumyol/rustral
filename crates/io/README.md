# rustral-io

Safetensors-based load/save for parameters and checkpoints.

Rustral is a Rust workspace for research and learning; see the [repository README](https://github.com/skumyol/rustral#readme) for install, examples, and status by backend.

For **experimental ONNX** emission of a single Linear layer (separate from Safetensors), see [`rustral-onnx-export`](../onnx-export/README.md).
