# rustral-hf

Optional Hugging Face Hub download/upload of weight files.

Rustral is a Rust workspace for research and learning; see the [repository README](https://github.com/skumyol/rustral#readme) for install, examples, and status by backend.

For **tensor name mapping** when loading Hub weights into Rustral modules (HF keys vs `NamedParameters`), see [`rustral-model-zoo`](../model-zoo/README.md).
