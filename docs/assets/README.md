# rustral logo assets

Included files:

- `rustral-logo-full.svg` — full logo lockup, transparent background
- `rustral-icon.svg` — icon-only version, transparent background
- `rustral-logo-full-mono.svg` — monochrome full logo
- `rustral-icon-mono.svg` — monochrome icon
- PNG previews, if exported successfully

Design notes:
- The icon uses a reduced Bauhaus geometry: circle, arc block, and layered bars.
- The stacked structure references rustral's explicit architecture: Backend / ForwardCtx / Parameter / Module / Runtime.
- The rust-orange accent nods to Rust without becoming too literal.
- SVG text remains editable. For production release, you may later outline the wordmark in Illustrator/Figma to avoid font dependency.
